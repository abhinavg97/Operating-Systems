g++ Assgn1-RMSes15btech11002.cpp -std=c++14 -o rm
g++ Assgn1-EDFSes15btech11002.cpp -std=c++14 -o edf

Make a inp-params.txt file or use the one attached.

In the inp-params.txt file 

First line consists of n, denoting the number of processes.
Then n lines follow, each of which should contain,

id t p k

where is the process id, t is the burst time required for that process, p is the period after which the process repeats, k is the number of times the process must be run.

The output is generated as mentioned in the pdf.
